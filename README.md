# Nepal Maternal Health Risk Dashboard (MERN Scaffold)

Structured MERN setup with Tailwind and Codespaces-ready configuration.

## Workspace Structure

- `backend/` Express + MongoDB-ready API scaffold
- `frontend/` Vite + React + Tailwind scaffold
- `shared/` shared constants and types placeholders
- `.devcontainer/` GitHub Codespaces configuration
- `.vscode/` editor recommendations

## Quick Start

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`

Backend health check: `http://localhost:5000/api/health`

## Notes

Feature code for the maternal risk modules is intentionally not implemented yet.
This setup focuses on structure, tooling, and developer environment.
